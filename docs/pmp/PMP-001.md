# PMP-001 — Plan Maestro del Proyecto

## BhagamAppsModular

**Versión:** 1.0
**Estado:** Aprobado
**Periodo de vigencia:** Junio 2026 – Junio 2027
**Propietario:** Dirección General del Proyecto

---

# 1. Propósito

El presente Plan Maestro del Proyecto (PMP-001) define la visión, alcance, objetivos, gobierno, prioridades y lineamientos generales para el desarrollo y evolución de BhagamAppsModular durante el periodo 2026–2027.

Este documento constituye la referencia principal para la toma de decisiones estratégicas del proyecto.

---

# 2. Visión

BhagamAppsModular será una plataforma de software libre especializada en la gestión integral de inventarios institucionales, proporcionando control, trazabilidad, custodia y administración de bienes mediante una arquitectura modular, escalable y mantenible.

---

# 3. Misión

Desarrollar y mantener una solución de gestión de inventarios que permita a instituciones administrar eficientemente sus bienes, responsables, ubicaciones, mantenimientos, movimientos y procesos de aprobación, garantizando trazabilidad y transparencia en la información.

---

# 4. Alcance

Los siguientes módulos existen para soportar la operación del sistema de inventarios:

```text
User
Apps
```

---

## Herramientas Internas

```text
CrudGenerator
```

CrudGenerator es una herramienta de desarrollo y no forma parte del producto entregado a los usuarios finales.

---

# 5. Exclusiones

BhagamAppsModular no contempla dentro de su alcance:

* Gestión académica.
* Gestión financiera.
* Gestión de nómina.
* Gestión documental general.
* ERP institucional completo.
* APPSisGOE.

---

# 6. Independencia de APPSisGOE

APPSisGOE es un proyecto independiente.

Ambos proyectos podrán compartir experiencia técnica, pero:

* Tendrán repositorios independientes.
* Tendrán documentación independiente.
* Tendrán roadmap independiente.
* Tendrán gobierno independiente.

No existe dependencia funcional entre ambos proyectos.
