# Propósito

Definir las metas institucionales base asociadas a los objetivos institucionales definidos en:

DDOM-GESTION-OBJ-001.

Las metas constituyen resultados medibles que orientan el mejoramiento institucional y servirán como punto de partida para cada institución usuaria de APPSisGOE.

---

# Principios

## Principio 1

Toda meta pertenece a un objetivo.

---

## Principio 2

Toda meta debe cumplir criterios SMART.

---

## Principio 3

Las metas propuestas son referenciales.

Cada institución podrá:

* Editarlas
* Ajustarlas
* Desactivarlas
* Crear nuevas
