# Propósito

Determinar el nivel de cobertura institucional alcanzado por APPSisGOE respecto a la estructura definida por la Guía 34 del MEN.

Este documento permite identificar:

* Componentes cubiertos.
* Componentes parcialmente cubiertos.
* Componentes sin cobertura.
* Prioridades institucionales futuras.
* Brechas funcionales.

---

# Alcance

La evaluación se realiza sobre:

* Gestión Directiva
* Gestión Académica
* Gestión Administrativa y Financiera
* Gestión de la Comunidad

---

# Niveles de Cobertura

## Nivel 0

No soportado.

No existe módulo.

No existen datos.

No existe funcionalidad.

---

## Nivel 1

Soportado parcialmente.

Existe módulo.

Existen datos.

La funcionalidad es limitada.

---

## Nivel 2

Soportado operativamente.

Existe módulo.

Existen datos.

Existen métricas operativas.

Existe uso institucional.

---

## Nivel 3

Soportado institucionalmente.

Existe:

* módulo
* métricas
* indicadores
* seguimiento
* evidencia
