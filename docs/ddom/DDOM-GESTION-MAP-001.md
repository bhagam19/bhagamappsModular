# Principio de Asignación

Toda meta deberá pertenecer a un único componente institucional principal.

Todo indicador deberá pertenecer a un único componente institucional principal.

Una meta podrá asociarse a uno o varios indicadores.

Un indicador podrá asociarse a múltiples metas.
